Adding underline
================

- ( ctrl + L )  + o,i,u

More here:
https://github.com/drmikehenry/vim-extline/blob/master/doc/extline.txt

Adding comment
==============

- <leader> + cc
- toggler : <leader> + c + space

More here : https://github.com/scrooloose/nerdcommenter

> NOTE : Leader is set to ',' (comma)
